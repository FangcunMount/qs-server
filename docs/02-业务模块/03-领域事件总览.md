# 领域事件总览

## 1. 结论

现行事件清单不在本文复制维护，唯一机器契约是 [`configs/events.yaml`](../../configs/events.yaml)。

## 2. 当前分组

| Topic 组 | 主要事件 | 语义 |
| --- | --- | --- |
| survey/model 生命周期 | `questionnaire.changed`、`assessment_model.changed` | best effort 变更通知 |
| evaluation 生命周期 | `answersheet.submitted`、`evaluation.*`、`interpretation.report.*` | 主链可靠出站 |
| plan task 生命周期 | `task.opened/completed/expired/canceled` | 任务变化通知 |

Redis Pub/Sub 的 `report_status_changed` 等是一次性信令，定义在 `configs/signals.yaml`，不属于领域事实。

## 3. 维护规则

新增或改名时必须同时更新 YAML 契约、代码 catalog/常量、发布与消费注册、契约测试和受影响模块文档。
