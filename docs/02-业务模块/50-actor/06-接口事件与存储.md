# Actor 接口事件与存储

## 1. 接口边界

Actor 提供受试者、操作者、关系和入口上下文相关能力。接口契约以 REST / gRPC 为准。

---

## 2. 事件边界

Actor 不产生 `footprint.*` 事件。入口解析和接入日志由 `behavior_journey_scan` 轮询，用作统计行为旅程的事实来源。

---

## 3. 存储边界

Actor 保存业务参与者视图、关系和必要的 IAM 引用，不保存 IAM 主身份完整模型。
