# Actor 接口事件与存储

## 1. 接口边界

Actor 提供受试者、操作者、关系和入口上下文相关能力。接口契约以 REST / gRPC 为准。

---

## 2. 事件边界

Actor 相关业务行为会形成统计 footprint：

| 事件 | 说明 |
| ---- | ---- |
| `footprint.testee_profile_created` | 新建受试者档案 |
| `footprint.care_relationship_established` | 建立服务或照护关系 |
| `footprint.care_relationship_transferred` | 转移服务或照护关系 |

---

## 3. 存储边界

Actor 保存业务参与者视图、关系和必要的 IAM 引用，不保存 IAM 主身份完整模型。
