# Plan 接口事件与存储

## 1. 接口边界

Plan 提供计划、任务、调度和入口相关能力。接口契约以 REST / gRPC 为准。

---

## 2. 事件边界

| 事件 | Delivery | 说明 |
| ---- | -------- | ---- |
| `task.opened` | `best_effort` | 任务开放 |
| `task.completed` | `best_effort` | 任务完成 |
| `task.expired` | `best_effort` | 任务过期 |
| `task.canceled` | `best_effort` | 任务取消 |

---

## 3. 存储边界

Plan 保存计划、任务、调度规则和跨模块引用，不保存答卷明细、测评分数或报告正文。
