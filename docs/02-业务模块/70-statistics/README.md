# Statistics

**本文回答**：Statistics 如何作为读侧统计与行为投影模块，消费业务事件并形成运营、计划、测评和行为指标。

---

## 1. 这个模块负责什么

Statistics 负责“如何看业务运行结果”：

- 机构概览。
- 测评服务漏斗。
- 行为足迹投影。
- 计划统计。
- 受试者、从业者、组织维度读模型。
- 查询缓存和最终一致性读侧。

---

## 2. 这个模块不负责什么

- 不维护问卷、答卷、测评、报告、计划的主事实。
- 不反向驱动核心业务状态。
- 不负责测评执行。
- 不负责报告生成。

一句话：**Statistics 不是事实源；Statistics 是读侧投影。**
