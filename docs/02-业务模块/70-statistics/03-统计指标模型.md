# 统计指标模型

## 1. 业务目标

把业务链路中的事实转换成运营、服务质量和计划完成情况可查询的指标。

---

## 2. 指标分组

| 指标组 | 来源 | 示例 |
| ------ | ---- | ---- |
| Survey 指标 | 答卷提交 | 提交数、有效数、无效数 |
| Assessment 指标 | 测评执行 | 创建数、完成数、失败数、完成率 |
| Report 指标 | 报告生成和查看 | 生成数、查看数 |
| Plan 指标 | 任务事件 | 任务数、完成数、过期数、完成率 |
| Journey 指标 | 行为 footprint | 入口打开、接入确认、建档、关系建立 |

---

## 3. 时间窗口

常见窗口包括：

- today
- 7d
- 30d

具体支持范围以代码和接口契约为准。

---

## 4. 关键规则

- 指标必须能回到事件或读模型来源。
- 不同口径要明确分母和时间窗口。
- 统计指标允许延迟，不应用作主链路成功条件。
