# Statistics 接口事件与存储

## 1. 接口边界

Statistics 提供统计查询接口和必要的读模型刷新能力。接口契约以 REST / gRPC 为准。

---

## 2. 事件边界

Statistics 主要消费：

| 事件 | 说明 |
| ---- | ---- |
| `footprint.*` | 行为旅程投影 |
| `answersheet.submitted` | 答卷提交指标 |
| `assessment.interpreted` | 测评完成指标 |
| `assessment.failed` | 测评失败指标 |
| `report.generated` | 报告生成指标 |
| `task.*` | 计划任务指标 |

---

## 3. 存储边界

Statistics 保存读模型、聚合指标、缓存和更新时间，不保存核心业务写模型。
