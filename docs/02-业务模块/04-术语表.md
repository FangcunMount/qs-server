# 术语表

## 1. 核心术语

| 术语 | 当前含义 |
| --- | --- |
| Questionnaire | 可发布、可版本化的问卷定义 |
| AnswerSheet | 用户已提交的作答事实 |
| Assessment Model | 产品/业务展示名；代码模块为 `modelcatalog` |
| DefinitionV2 | 当前测评模型定义的统一语义载体 |
| Published Model | 运行时可读取的已发布模型快照；不是可编辑草稿 |
| Assessment | 一次测评执行的业务标识与生命周期 |
| EvaluationRun | 一次执行尝试 |
| Outcome | Evaluation 已可靠提交的评分/分类事实 |
| ReportGeneration | Interpretation 中的报告生成聚合 |
| InterpretationRun | 一次报告生成尝试 |
| InterpretReport | 不可变报告成品 |
| Outbox | 业务事务内记录、事务外可靠发布的事件机制 |
| Signal | 可丢失、可由查询/TTL 兜底的一次性唤醒 |

## 2. 易混边界

- ProductChannel 是产品分类，不应直接决定执行路由。
- `modelcatalog` 管模型资产，`evaluation` 管执行，`interpretation` 管报告。
- 事件是业务事实，信令是在线协作提示。
