# Survey 分层架构与代码索引

## 1. 分层总图

```text
transport
  REST questionnaire/answersheet handlers
  gRPC QuestionnaireService/AnswerSheetService
      ↓
application/survey
  questionnaire lifecycle/content/query
  answersheet submission/management/scoring
      ↓
domain/survey
  Questionnaire / Question / SubmissionSpec
  AnswerSheet / Answer / SubmissionContext
      ↓ ports
infra + container
  Mongo repositories/read models
  cache decorator / ruleengine / transaction / outbox profile
```

跨模块 AnswerSheet → Assessment journey 位于 `application/journey/assessmentintake`，不属于 Survey application。
