# Survey

**本文回答**：Survey 模块如何作为作答事实层，负责问卷定义、题目结构、答案校验和 `AnswerSheet` 事实沉淀。

---

## 1. 这个模块负责什么

Survey 负责“用户答了什么”这一类事实：

- `Questionnaire`：问卷定义、基础信息、发布状态、版本。
- `Question`：题目、题型、选项、必填和展示规则。
- `SubmissionSpec`：提交规格和填报约束。
- `AnswerSheet`：一次答卷提交后的事实记录。
- `AnswerItem` / `AnswerValue`：单题答案结构。
- `answersheet.submitted`：答卷提交后的可靠出站事件。

---

## 2. 这个模块不负责什么

Survey 不负责：

- 测评模型资产和发布快照。
- 计分规则权威。
- 一次测评执行状态机。
- 解释模型和最终报告。
- 周期任务生命周期。
- 统计指标聚合。

一句话：**Survey 只保证“答了什么”准确，不解释“得多少分”或“代表什么”。**
