# 题目校验与答案结构

## 1. 业务目标

确保用户提交的答案与问卷题目结构一致，使后续测评执行可以信任 `AnswerSheet` 的基础格式。

---

## 2. 题目模型

| 概念 | 说明 |
| ---- | ---- |
| `Question` | 题目实体 |
| `QuestionType` | 题型，例如单选、多选、文本、数字、分段等 |
| `Option` | 选择题选项 |
| `Required` | 必填约束 |
| `ValidationRule` | 题型相关校验规则 |

---

## 3. 答案结构

| 概念 | 说明 |
| ---- | ---- |
| `AnswerItem` | 单题答案 |
| `QuestionID` | 答案对应的题目 |
| `Value` | 结构化答案值 |
| `RawValue` | 必要时保留原始提交值 |

---

## 4. 校验规则

| 规则 | 说明 |
| ---- | ---- |
| 问卷引用 | 答案必须属于当前问卷版本 |
| 必填 | 必填题必须存在有效答案 |
| 唯一性 | 同一题不能重复提交互相冲突的答案 |
| 题型匹配 | 单选、多选、文本、数字按题型校验 |
| 选项合法 | 选择题答案必须来自有效选项 |

---

## 5. 边界

题目校验不做：

- 分数计算。
- 因子计算。
- 等级判定。
- 解释文案选择。
- 报告 section 渲染。

这些能力分别归属 `evaluation` 和 `interpretation-model`。
