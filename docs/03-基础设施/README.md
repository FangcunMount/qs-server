# 基础设施

本组文档介绍 `qs-server` 的基础设施层实现，重点回答五个问题：

- 领域事件如何发布、路由和消费
- 数据为什么分布在 `MySQL / MongoDB / Redis`
- 缓存、限流、排队和背压分别放在哪里
- 系统如何接入 IAM、JWT、JWKS 和 mTLS
- 运行配置如何从 YAML 与 Options 进入各个进程

如果你第一次读这组文档，建议顺序是：

1. [01-事件系统.md](./01-事件系统.md)
2. [02-存储模型.md](./02-存储模型.md)
3. [03-缓存与限流.md](./03-缓存与限流.md)

5. [05-配置体系.md](./05-配置体系.md)
