## 2. 30 秒结论

`qs-server` 是从 ADHD 医学量表场景生长出来的多模型测评平台，也是以 `qs-apiserver` 为主业务中心、由 `collection-server` 和 `qs-worker` 协作的三进程系统。业务事实按 `survey -> modelcatalog -> evaluation -> interpretation` 分层，`actor`、`plan`、`statistics` 提供参与者、编排与读侧支撑。架构首先服务多模型扩展和可靠异步执行，再通过 BFF/IAM 边界、缓存与过载保护满足接入和运行质量要求。
