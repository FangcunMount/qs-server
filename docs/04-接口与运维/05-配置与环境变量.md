# 配置与环境变量

## 1. 事实源

进程配置位于 [`configs`](../../configs/)；环境变量约定见 `configs/env/README.md`。本文不复制完整默认值。

## 2. 配置分层

- `*.dev.yaml`：本地/开发默认；
- `*.prod.yaml`：生产基线；
- 环境变量：凭证、地址或部署差异覆盖；
- 运行时治理：只能调整明确暴露、带校验和审计的能力。

## 3. 变更检查

配置字段变化时同步：配置结构、默认文件、环境变量说明、装配代码、校验测试和运维文档。敏感值不得进入仓库或日志。

## 4. 高风险项

数据库/MQ/Redis 地址、服务认证、队列容量、超时重试、租约续期、后台任务开关和治理能力必须在发布前核对生产配置。
