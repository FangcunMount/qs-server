# 量表统计（含授权落地要点）

## 授权落地（总流程）

* 定版（名称/版本/语言/分表/题目数）→ 明确使用边界（内部/对外/收费/导出/分享）→ 找版权方 → 申请“复制题目+电子化+商业”许可 → 签约留档 → 系统开关落地（导出/分享/API/水印/审计）→ 年度复审

## 高风险功能（最容易踩线）

* 题库化（可搜索/可复制/可下载题干）
* 导出/打印/分享链接
* 开放 API 拉取题目/报告全文
* 嵌入常模/阈值/解释文本
* 翻译/改写/删减条目后仍用原量表名

## 一级类别体系 v1（主类）

> 归类原则：每个量表只选 1 个**主类**；其余信息用**标签**表达（最多 3–5 个）。

| 主类                | 收纳范围（主测量对象）                           | 典型例子                            |
| ----------------- | ------------------------------------- | ------------------------------- |
| ADHD              | ADHD 核心症状、课堂行为、外化/对立相关（以注意缺陷/多动冲动为中心） | SNAP-IV、Vanderbilt、Conners、IOWA |
| 抽动障碍              | 运动/发声抽动严重度、损害与随访                      | YGTSS                           |
| 感统                | 感觉处理/感觉统合/动作计划/社会参与等感知-运动构念           | SPM、感觉统合评定                      |
| 执行功能              | 抑制/工作记忆/计划组织/转换/情绪控制等执行功能与自我调节        | BRIEF-2、执行功能他评                  |
| 心理健康              | 焦虑/抑郁/广谱心理症状、儿童行为情绪谱、压力与家庭过程指标        | SCARED、SCL-90、CBCL、FASA、PSS     |
| 认知筛查（建议命名：神经发育筛查） | 以神经发育/ASD 风险筛查为主（你库里目前主要落在 ASD 早筛）    | M-CHAT-R/F（你条目里疑似）              |
| 慢性病管理             | 躯体症状评分、喂养/过敏/消化等“风险提示+趋势随访”工具         | CoMiSS                          |
| 生活质量              | 功能受损/日常困难/结局（outcome），强调“能否正常生活学习社交”  | WFIRS-P、QCD                     |

### 标签体系 v1（建议）

* **阶段**：筛查、深评、随访、结局
* **填报人**：家长评、教师评、自评、临床评定
* **主题**：共病、功能、家庭系统、压力、广谱、简版、婴幼儿、学龄
* **状态**：需定版、自定义
