# Archive

`docs/_archive` 是 `qs-server` 文档系统里的**历史层**，用于存放旧设计稿、阶段记录和已经退出当前真值层的材料。

## 定位

- 本目录**不属于现行真值层**
- 本目录默认**不进入现行阅读主路径**
- 本目录默认**不参与 `make docs-hygiene`**

现行阅读和维护应优先回到：

- `docs/00-总览`
- `docs/01-运行时`
- `docs/02-业务模块`
- `docs/03-基础设施`
- `docs/04-接口与运维`
- `docs/05-专题分析`
- `docs/06-宣讲`

## 使用规则

- archive 只用于补充历史背景，不直接提供当前结论
- 从 archive 回迁内容前，必须重新核对源码、契约和配置
- archive 中文档应在开头明确标注“历史资料 / 非当前实现”

## 当前归档主题

- `docs/_archive/03-基础设施/`
- `docs/_archive/05-专题分析/`

## 维护提示

- 不要把 archive 当作第二真值层
- 不要在现行文档里把 archive 作为必经入口
- 如需新增归档文档，优先保证“为什么归档”和“现行入口在哪里”写清楚
