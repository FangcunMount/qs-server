# 文档归档区

> 说明：本目录当前仅作为临时信息补充和历史参考，后续会删除；不保证内部链接、代码锚点和路径仍然有效。

## 使用方式

- 归档区默认不作为“当前实现”的入口，也不作为接口、运行时和模块边界的最终依据。
- 阅读当前系统时，请优先从 [../README.md](../README.md) 和新的 `00-总览` 到 `04-接口与运维` 进入。
- 归档文档保留原始目录和文件名，便于回溯；其中的描述、路径、示例代码和相对链接可能已经过时。
