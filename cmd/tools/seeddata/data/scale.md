# 量表统计（含授权落地要点）

## 授权落地（总流程）

* 定版（名称/版本/语言/分表/题目数）→ 明确使用边界（内部/对外/收费/导出/分享）→ 找版权方 → 申请“复制题目+电子化+商业”许可 → 签约留档 → 系统开关落地（导出/分享/API/水印/审计）→ 年度复审

## 高风险功能（最容易踩线）

* 题库化（可搜索/可复制/可下载题干）
* 导出/打印/分享链接
* 开放 API 拉取题目/报告全文
* 嵌入常模/阈值/解释文本
* 翻译/改写/删减条目后仍用原量表名

## 一级类别体系 v1（主类）

归类原则：每个量表只选 1 个**主类**；跨界信息用**标签**表达（建议 ≤5 个）。

* **ADHD**：注意缺陷/多动冲动/对立外化相关（症状、课堂行为、短量表随访）。
* **抽动障碍**：运动/发声抽动严重度与功能损害（随访监测）。
* **感统**：感觉处理/感觉统合/动作计划与参与（干预评估/随访）。
* **执行功能**：抑制、转换、情绪控制、工作记忆、计划组织等自我调节能力。
* **心理健康**：焦虑、抑郁、广谱心理症状、儿童行为情绪谱、压力与家庭过程指标。
* **认知筛查（建议命名：神经发育筛查）**：发育/ASD 等神经发育风险筛查与评定。
* **慢性病管理**：躯体健康工具（如喂养/过敏/消化等症状评分）用于风险提示与随访。
* **生活质量**：功能受损、日常困难、参与度/角色表现（学习/家庭/社交/生活技能等结局）。

## 标签体系 v1（仅保留“主题标签”）

你说得对：**标签只保留“主题”**会更干净，且不会把“阶段/评定者/人群”混进标签体系里。

* **主题标签（多选，建议 ≤5）**：

  * 共病/对立外化（ODD、对立、外化）
  * 攻击/品行（攻击、品行问题、冲动攻击）
  * 功能结局/生活质量（功能受损、日常困难、表现/参与）
  * 焦虑
  * 抑郁
  * 广谱心理症状（如 SCL-90 这类症状谱系）
  * 广谱行为情绪（如 CBCL 这类行为情绪谱）
  * 家庭系统/顺应（如 FASA）
  * 压力/应激（如 PSS、家长压力）
  * ASD筛查/评定（神经发育相关）
  * 喂养/过敏/消化（如 CoMiSS）
  * 社会能力/同伴（社交/同伴/社会参与）
  * 情绪调节（情绪控制/调节能力）

> 说明：**阶段、评定者、人群不再做标签**，而是作为结构化字段维护（例如 `stage`、`rater`、`age_group`）。
